02. 이벤트 핸들러 프로퍼티 방식

        DOM 요소는 우리가 취득을 하고 출력을 했을때,
        수많은 프로퍼티들이 존재 했다. 그중 이벤트 관련 프로퍼티는 
        on 접두사 + 이벤트 타입 형식으로 되어 있으며, 함수를 집어 넣어주면 된다.

        어트리뷰트 방식과 비교를 했을때 HTML 과 javaScript 언어가 분리가 되었다.
        따라서 우린는 자바스크립트 언어는 따로 js 파일에 저당를 해두고 필요시 head 영역에서 읽어드리면
        html 파일은 html 요소만 있는 파일로 만들수 있다.
