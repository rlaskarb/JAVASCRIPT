01. innerHTML

        innerHTML property는 getter,setter 가 존재하는 property이다.
        요소 노드의 HTML 마크업 언어를 취득하거나 변경한다.
        textContent는 HTML 마크업을 무시하고 text
        문자열을 반환해준다.
태그 엘리먼트 값 읽거나 변경 시innerHTML
        innerHTML 을 사용하면 DOM 조작이 직관적이고 
        간단하다는 장점이 존재한다. 하지만 사용자가 입력한 데이터를 그대로
        innerHTML 에 넣게 되면 HTML을 아는 악성 유저는 이를 이용해서 에러를 발생시킬 수 있다.
