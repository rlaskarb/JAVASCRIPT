01. 요소 노드의 텍스트 조작

nodeValue

        nodeValue 라는 property는
        getter,setter 가 모두 존재하는 property이다.
        nodeValue property 참조시 node 객체의 값
        (텍스트 node의 텍스트)을 반환해준다.
        여기서 주의할 점은 텍스트가 아닌
        요소 node인 경우에는 null 을 반환해준다.
변경전

textContent

        textCOntent property 도 getter 와 setter 가 모두 존재하는 property이다.
        하지만 위에서 nodeValue 와 다른점은 요소 node 텍스트와 모든 자손node 의 
        텍스트를 모두 취득하거나 변경할 수 있다.
