01. open , alert , confirm , prompt 메소드 

        자바스크립트 언어는 브라우저 상에서 동작하는 언어이다.
        자바스크립트가 web 상에서 동작을 할때
        사용할 수 있는 기능이 있다.
        web 상에서 가장 최상의 객체는 window 객체이다.
        window
            - DOM (document 등등)
            - BOM (location , navigator 등등)
            - JavaScript(Array, Function , Object 등등)
        최 상단의 window 객체는 브라우저 창을 대변하고
        브라우저를 제어할 수 있는 메소드를 제공한다.
        window 는 전역 객체이기 때문에 생략이 가능하다.
        ex -> window.alert() -> alert()

open

        window.open(url,name,params) 메소드로 새로운 창을 열수 있다.

alert

        깃 허 브 컨트로 시

confirm

        깃 허 브 컨트롤 시

prompt 

        window.prompt(message , default)
        message : 사용자가 어떤 값을 입력해야 되는지 안내문구
        default : 택스트 입력 필드에 기본 값(문자열)
        반환 값은 사용자 입력 값이며, 취소를 누르거나 esc 키를 누르게 되면 null 값 리턴
